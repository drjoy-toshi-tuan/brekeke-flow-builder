### 抽出した要件サマリー

- **用件 4 分類 + DTMF 対応**（1=変更, 2=キャンセル, 3=確認, 4=予約。発話派生語は OpenAI で集約）
- **症状確認ゲート**（緊急発話 → AI 受付対象外として代表電話案内へ即離脱）
- **当日予約確認**（予約変更/キャンセル/確認のみ通過、当日希望 → 代表案内）
- **通院中確認**（予約=新規/再診ルートのみ通過、再診 vs 初診を classification に上書き）
- **紹介状確認**（初診のみ通過、なし → 選定療養費案内＋代表電話で離脱）
- **診療科聴取**（19 個の確定診療科 + 救急外来 + 内科単独 + 5 つの曖昧分岐トリガー + 登録なし）
- **AI 非対応診療科**（救急外来 / 歯科口腔外科×変更キャンセル / 心臓リハビリ外来 → 病院代表 098-878-0231 ×3）
- **代表案内系**（症状あり / 本日予約 / 紹介状なし / 診療科不明 / 聴取失敗 → 予約センター 0120-979-706 ×3）
- **氏名聴取・生年月日聴取（復唱あり）・電話番号聴取（復唱あり、phonetype 判定）の 3 サブフロー**
- **予約日（datetime）/理由（text）/希望日（text）**: 予約変更/キャンセルのみ予約日・理由を聴取、予約変更のみ希望日も追加聴取
- **完了 TTS 3 種**:
  - 予約変更/キャンセル/再診/初診 → 折り返し+SMS両案内 (smsFlag=1)
  - 予約確認×携帯 → SMSのみ (smsFlag=1)
  - 予約確認×固定/その他 → 折り返しのみ (smsFlag=-1)
- **電話番号案内の 3 回繰り返し読み上げ**（Gen1 案内回数カウント踏襲、文言を縮めない）
- **acceptance_times**: 月-金 8:30-17:30 / 土 8:30-12:30 / 日祝・年末年始休止
- **status 値マッピング**: 1=未処理 / 2=代表案内 / 3=聴取失敗 / 6=時間外（非通知も 2 を共用）
