### 抽出した要件サマリー

- **用件 4 種** (新規予約 / 予約変更 / 予約キャンセル / 予約確認) + **CT/MRI 検査** (新規予約の特殊レーン) を Pattern A の OpenAI hearing 1 本で識別
- **DTMF 1=新規予約 / 2=予約変更 / 3=予約キャンセル / 4=予約確認** を冒頭ガイダンスで明示せず裏で受付 (Gen1 用件.yaml の DTMF 分岐踏襲)
- **echo_back (復唱)**: 用件確認 + 診療科聴取の 2 ブロックのみ (氏名/生年月日/電話/診察券は個人情報サブフロー内で完結)
- **診療科 38 種**: 分岐なし 33 種 + 分岐あり 5 種 (呼吸器・消化器・脳神経・産婦人科・内分泌) を leaf 段で扱う方針
- **クローズ診療科 5 種**: 血液内科 / 精神科 (新規予約のみ) / 子どものこころ診療部 / 海外渡航者ワクチン外来 (受診歴あり再診のみ通常フロー) / 移植外科
- **頭痛外来の特殊扱い**: 紹介状なしの場合、初診時選定療養費 7,700 円の自己負担案内のみ → そのまま受付続行 (他診療科の紹介状なしは代表案内へ)
- **海外渡航者ワクチン外来 専用転送**: 受診歴 = 再診 + 入電時刻 13:00〜16:00 のみ 0263-37-3475 へ Blind Transfer。それ以外は代表案内 / 時間外案内 / 初診不可案内
- **ペースメーカー有無確認**: CT/MRI 検査時のみ前段に「(CT|MRI) 検査を受診される方へ」アナウンスで起動
- **産科のみ二段聴取**: 出産予定日 + 妊娠週数 (わからない発話可)
- **新規予約の追加質問**: 診療科 → (産科 → 出産予定日; それ以外 → 受診歴 → 紹介状 → 紹介元医療機関名)

- **予約キャンセルの追加質問**: 診療科 → 予約日 → キャンセル理由 (体調変化時のみ症状追加聴取) — 移管ノートに従い変更希望日と残薬確認はスキップ

- **終話 8 通り** (4 用件 × 携帯/その他)。CT/MRI は新規予約と同じ終話 TTS にロールアップ
