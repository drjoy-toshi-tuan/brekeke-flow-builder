### 診療科マッピングの異常パターン確認

プロンプト「歯科口腔外科」セクション（line 229-231）:
```
鹿
イカ
いか
```

**確認**: これらは STT 誤変換パターンとしての記載と推定される。
- 「鹿」（しか）= 「歯科」（しか）の同音異字
- 「イカ」「いか」 = 不明（要確認）

設計書では AmiVoice 辞書に「歯科 しか」「歯医者 はいしゃ」と定義されているが、「イカ」「鹿」の STT パターンは記載されていない。

**判定**: これは「設計意図の不明確さ」というより、「プロンプトの曖昧な設計」。STT 辞書と整合性があるか、または意図的な誤変換対策なのか不明。セキュリティ上の問題ではなく、**精度上の懸念**となるため、別途確認が必要。
