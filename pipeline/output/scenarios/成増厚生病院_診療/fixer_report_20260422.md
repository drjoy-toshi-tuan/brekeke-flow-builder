- **修正担当**: 設計段階（prompter は設計書に準拠）
- **対象モジュール**: OpenAI_受診歴確認（L215-220）、OpenAI_紹介状（L1085-1090）、OpenAI_本人確認（L1310-1315）

- **重大度**: Info（設計書に明示されているため prompt は正確だが、設計段階で再検討推奨）
- **修正方針**: 設計段階での再検討。または「肺」を含む文脈判定（「肺のことで」で「肺」単体を無視）を追加
