### 抽出した要件サマリー

- 冒頭 DTMF 2択（本日受診=1 / 翌日以降=2）で大分岐
- 本日受診: 診療科 → 受診歴 → 紹介状 →(なし)選定療養費(7,000円) → 症状 → 希望日程 → 個人情報 → 終話①
- 翌日以降: 用件 DTMF 4択（予約/変更/キャンセル/問い合わせ）
  - 予約: 予約内容 → 予約の種類(診察予約/区健診/妊婦初診/それ以外)。妊婦初診は出産予定日・和通分娩・区市町村・既往歴を追加 → 終話③
  - 変更: 診療科 → 現在予約日 → 希望日程 → 個人情報 → 終話①
  - キャンセル: 診療科 → 現在予約日 → キャンセル理由 → 個人情報 → 終話①
  - 問い合わせ: 確認内容(FAQ) → 終話②
- 救急科は AI 非対応(×) → END_救急科（代表案内）
- 個人情報: 氏名 / 年齢 / 生年月日 / 診察券番号 / 連絡先電話番号（サブフロー）
- FAQ 約23件（診察券紛失・紹介状・最寄駅・駐車場・送迎バス・診療時間 等）
