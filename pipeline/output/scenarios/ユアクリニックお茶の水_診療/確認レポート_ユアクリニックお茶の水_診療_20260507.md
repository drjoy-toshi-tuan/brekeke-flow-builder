- **平日**: 8:30 前=時間外 / 8:30〜11:00=代表転送 / 11:00〜15:00=AI / 15:00〜17:00=代表転送 / 17:00〜19:00=AI / 19:00 以降=時間外
- **土曜**: 8:30 前=時間外 / 8:30〜11:00=AI / 11:00〜13:30=代表転送 / 13:30〜15:00=AI / 15:00〜17:00=代表転送 / 17:00 以降=時間外
- **祝日**: 8:30 前=時間外 / 8:30〜11:00=AI / 11:00〜13:00=代表転送 / 13:00 以降=時間外
- **2026 年 GW 特別日付（year-specific）**:
  - 4/29 (水・祝): 8:30〜16:00=代表転送 / それ以外=時間外
  - 5/1 (金): 通常曜日扱い（=平日ルール）
  - 5/2 (土): 通常曜日扱い（=土曜ルール）
  - 5/3 (日・祝): 全日時間外
  - 5/4 (月・祝): 8:30〜13:00=代表転送 / それ以外=時間外
  - 5/5 (火・祝): 8:30〜16:00=代表転送 / それ以外=時間外
  - 5/6 (水・祝): 8:30〜13:00=代表転送 / それ以外=時間外
- **代表電話転送（Blind Transfer）**: 一部時間帯では AI 受付ではなく人スタッフへブラインド転送。転送先電話番号は properties マネージャー入力枠
