### 抽出した要件サマリー

- **メインフロー聴取順序**: 氏名 → 診察券番号（7桁）→ 生年月日 → 連絡先電話番号 → 用件確認 → 用件別フロー → 終話分岐
- **用件 4 種**: 確認 / 変更 / キャンセル / その他（DTMF 1=確認, 2=変更, 3=キャンセル, 4=その他、+ 音声並行受付）
- **用件別フロー**:
  - 確認: 確認内容（自由テキスト） → 終話
  - 変更: 現在の予約日 → 希望日（自由テキスト） → 終話
  - キャンセル: 現在の予約日 → キャンセル理由（自由テキスト） → 終話
  - その他: 確認内容（自由テキスト、確認用件と合流） → 終話
- **終話**: 用件 4 種で TTS 文言が異なる（4 通り）。SMS 送信は 4 用件すべてで `smsFlag=1`（Gen1 元資料に「ショートメールにてご連絡」と全用件で記載）。携帯/固定の SMS 出し分けは Gen1 になし。
- **受付時間**: 平日 8:30〜17:00。土日祝・GW 期間（5/2〜5/6）は時間外（GW 期間は Dr.JOY 側カレンダーで休業日扱い推奨）。
- **個人情報サブフロー**: 氏名 / 生年月日 / 電話番号 / 診察券番号 の 4 本（テンプレ完全コピー、CLAUDE.md Rule 9 準拠）。
- **echo_back（復唱）対象**: 用件確認 / 診察券番号（サブ内蔵）/ 生年月日（サブ内蔵）/ 連絡先電話番号（サブ内蔵）。診療科聴取は本フローには無い（用件と直接の紐付けがあり、診療科聴取が無いまま用件別フローに進む構成）。
- **代表電話番号**: 096-351-8051（聴取失敗時の END_聴取失敗 案内に組み込む）
