- **診療科**: 12 科（循環器・心臓血管特化）+ 緊急救急 + 未対応 6 科 + 登録なし
  - 通常: ペースメーカー外来・児童精神科・循環器内科・心療内科・糖尿病内科・泌尿器科・不整脈科・血管外科・呼吸器外科・心臓血管外科・整形外科・脳神経外科
  - 未対応: セカンドオピニオン外来・フットケア外来・外来リハビリ・健診・心血管センター・予防接種
  - 緊急救急: END_緊急救急 で代表電話誘導
- **変更/キャンセル/確認ルート**: 予約日聴取 + 理由 / 内容（フリーテキスト）+ キャンセルのみ次予約希望日確認
- **個人情報聴取**: 氏名 → 生年月日 → 電話番号 → （新規以外なら診察券番号）
- **終話**: classification (4) × phonetype (2) = 8 終話 + 異常系（当日予約 / 緊急救急 / 未対応診療科 / 聴取失敗 / 非通知 / 時間外）の計 14 終話
- **SMS 分岐**: phonetype=1（携帯/ip050）→ smsFlag=1、phonetype=2（固定/IP/その他）→ smsFlag=-1
- **status 値**: 1=受付完了 / 2=代表案内 / 3=聴取失敗 / 6=時間外（Gen1 status=0/5 は使用禁止）
