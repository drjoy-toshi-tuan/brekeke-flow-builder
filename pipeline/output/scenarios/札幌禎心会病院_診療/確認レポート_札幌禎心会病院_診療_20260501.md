### 抽出した要件サマリー

- **冒頭**: 営業時間判定（年末年始12/30-1/4 / 平日9-17 / 土9-12 / それ以外時間外）+ 施設冒頭挨拶
- **当日予約不可**: 「本日の予約に関する内容ですか？」Yes→当日代表案内終話 / No→用件聴取
- **2段階用件分類**: 第1段「予約 vs 情報提供書」(復唱あり) → 予約系の場合は当日受診確認 → 第2段「診察予約/予約変更/予約キャンセル/予約確認」4択(復唱あり)
- **新患確認**: 診察予約選択時のみ「初診ですか？」を聞き、初診/再診で classification を上書き
- **紹介状有無**: 新規予約_初診ルートのみ。なしの場合「初診時選定療養費1,100円全額自己負担」アナウンス
- **診療科**: 通常診療科25種 + 代表案内診療科3種(がんセカンドオピニオン外来/救急/血管内留置デバイス外来)。代表案内ヒット時は終話、5回NG+2回再質問でも取れない場合は END_診療科聞き取りNG
- **予約日聴取**: 予約変更/キャンセル時のみ。月日のみ（西暦不要）、過去日付なら+1年自動補完、DTMF 4桁対応
- **確認内容聴取**: 予約確認ルートのみ。フリーテキスト
- **情報提供書ルート**: 医療機関名フリーワード → 診療科宛先フリーワード
- **個人情報聴取**: 氏名(subject=patient) → 生年月日(復唱あり、subject=patient) → 連絡先電話番号(incoming-classifier型、復唱あり、固定取得時は緊急時携帯番号オプション)
- **診察券番号**: 予約変更/キャンセル/予約確認/情報提供書ルートのみ。新規予約(初診/再診)では聴取しない
- **smsFlag分岐**: 携帯あり→「3診療日以内に折り返し電話 or SMS」(smsFlag=1) / 固定のみ→「3診療日以内に折り返し電話」(smsFlag=-1)
- **代表案内**: 011-712-1131 を2回読み上げ（「電話番号は011-712-1131です。繰り返します。電話番号は011-712-1131です。」）
