### 判定した作業種別

- **作業種別**: Gen1→Gen3 移管
- **施設名**: 近森病院
- **シナリオ名**: 診療

### 抽出した要件サマリー

- **メインフロー聴取順序**: 用件確認 → 診察券番号 → 受診者名 → 生年月日 → 診療科 → 用件分岐 → 用件別フロー → 連絡先電話番号 → 最後の質問 → 終話分岐
- **用件 4 種**: 変更 / キャンセル / 予約 / 確認（DTMF 1-4 + 音声並行受付）
- **用件別フロー**:
  - 変更: 予約日 → 連絡先
  - キャンセル: 予約日 → キャンセル理由 → 連絡先
  - 予約: 予約希望日（自由テキスト） → 連絡先
  - 確認: 確認内容（自由テキスト） → 連絡先
- **終話**: 用件 4 種 × 電話種別（携帯/固定）= 8 通り。携帯=SMS 送信あり、固定/その他=SMS なし
- **24 時間受付ボット**: 営業時間チェック不要
- **個人情報サブフロー**: 氏名 / 生年月日 / 電話番号 / 診察券番号 の 4 本（テンプレ完全コピー）
- **echo_back（復唱）対象**: 用件確認 / 診療科 / 予約日 / 生年月日（サブ内蔵）/ 電話番号（サブ内蔵）
