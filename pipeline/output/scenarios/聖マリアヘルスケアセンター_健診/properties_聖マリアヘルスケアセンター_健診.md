## 補足

- `復唱_予約_個人` / `復唱_予約_企業` など `_個人` / `_企業` サフィックスが付くモジュールは、設計書の共通テキストをそれぞれに適用しています
- `予約日_復唱_*` の `{reservationDate}` はBrekekeのコンテキスト変数です。そのままコピーしてください（TTSエンジンが実行時に展開します）
- リトライモジュール（Speech Retry Counter）の `prompt_true` / `prompt_false` はIVRプロパティには記述しません。フローJSON内の params に直接記述済みです
- RAGサブフロー（`相談_問合せ` / `相談_問合せループ`）のデフォルトテキストはCLAUDE.mdの設計原則（パターン3）に基づくデフォルト文言を設定しています。内容を変更する場合は上記プロパティ欄を編集してください
