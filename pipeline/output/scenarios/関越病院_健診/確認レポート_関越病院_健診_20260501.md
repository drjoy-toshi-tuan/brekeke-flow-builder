### 抽出した要件サマリー

- **フロー概要**: 関越病院 健診センター専用 AI 電話。新規予約・変更/キャンセル・折り返し連絡・その他問合せの 4 用件を分類し、用件別の聴取（コース/予約希望日/変更内容/問い合わせ等）+ 共通聴取（氏名・生年月日・電話番号）を行うフロー
- **聴取項目（用件別）**:
  - 共通: 用件確認（DTMF + 音声、4 分岐、復唱あり）
  - 新規: 希望コース（DTMF + 音声、4 分岐、復唱あり）→ 予約希望/人数（自由テキスト）→ 受診者名 → 生年月日 → 連絡先電話番号
  - 変更/キャンセル: 現在の予約日（datetime、復唱あり）→ 変更内容判別（DTMF + 音声、3 分岐、復唱あり）→ 変更詳細 / キャンセル理由 → 変更/キャンセル理由 → 受診者名 → 生年月日 → 連絡先電話番号
  - 折り返し: 折り返し案内 → 受診者名 → 生年月日 → 連絡先電話番号
  - その他: 当日予約確認 → (Yes→代表電話案内終話 / No→問い合わせ内容 → 担当者名 → 連絡先電話番号)
- **分岐構造**: Pattern A（用件直後分岐）+ Pattern B（終話前 ContextMatchRouter で classification × changeContent の組合せ分岐）
- **施設固有ルール**: (a) 着信元番号 05017070320 / 08044220245 から発信 → 「市の乳がん／胃がん検診は代表番号へ」案内（augment 暫定実装）。(b) 当日予約は「その他お問い合わせ」用件配下（main.yaml 仕様）に配置
- **終話パターン**: 5 種正常系（新規受付完了 / 変更受付完了 / キャンセル受付完了 / 折り返し連絡受付 / 問い合わせ受付）+ 2 種案内系（代表電話案内 / 市の健診案内）+ 異常系 3 種（非通知 / 時間外 / 聴取失敗）
- **営業時間**: 月〜金 8:30〜17:30 / 土 8:30〜12:00 / 日祝休 / 12/30 12:00〜1/4 年末年始休（Gen3 では acceptance_times に標準セット、年末年始は Dr.JOY スケジュールで吸収）
- **特殊仕様**:
  - 3/2〜3/31 のブレケケ移行告知（050-1808-1225）は **一時運用済みのため除外**（移管ノート明記）
  - 担当者名/受診者名は 1 ステップで分岐 TTS（受診者名 vs 担当者名）。Gen3 PatientName サブフロー 1 枠ルール準拠で patientName サブフロー（受診者名）+ callerName hearing（その他用件の担当者名）に分離
  - 生年月日は和暦/西暦自動判別（DOB Re-confirmation で吸収）
  - 電話番号は incoming-classifier 携帯/固定判別 + Phone Normalization + 復唱（標準サブフローで吸収）
  - その他ルートのみ生年月日聴取をスキップ（Gen1「用件の分岐_2」仕様）
- **企業団体名聴取**: Gen1 に `END_企業団体名聴取` サブシナリオあり（`企業団体名聴取.yaml`）だが main.yaml から到達経路なし。**director 判断で除外**（移管ノート 25 行目）
