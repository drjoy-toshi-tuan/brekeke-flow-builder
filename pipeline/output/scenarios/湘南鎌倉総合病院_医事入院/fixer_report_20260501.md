#### W-001: 再診キーワード冗長性 <prompter>
- **重大度**: Warning
- **対象**: OpenAI_初診確認（L63-80）
- **問題**: 再診キーワードリストが設計書（step_details L429-430）の記載を大幅に超過
  - 設計書: 「いいえ / いえ / じゃない / 違う / 受診したことがある / 既に受診」
  - プロンプト: 上記に加え「ではない」「ちがう」「受診したことあり」「すでに受診」「通院している」「かかったことがある」「来たことがある」「再診」「二回目」「2回目」を追加
- **指摘**: SKILL ガイド（表記ゆれ禁止）に基づく表記ゆれ追加は許容ですが、以下は設計書に根拠がない：
  - 「通院している」（単なる受診 vs 継続受診の区別）
  - 「かかったことがある」「来たことがある」（ユーザーが「診察にかかったことがある」と答える確率は低い）
  - 「再診」「二回目」「2回目」（ユーザーが「再診です」と言う確率は低い）
- **修正方針**: 設計書の明示的キーワードのみ使用。不必要な拡張は削除。表記ゆれ（「ですだけ」「ちがう」等）のみ追加を許容すべき。
