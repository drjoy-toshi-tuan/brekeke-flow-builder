### 抽出した要件サマリー

- **24 時間稼働**: 営業時間チェック不要。GW 期間（4/30 17:00 ～ 5/7 7:59）のみ Script (custom, calendar) で休診案内 → 切断
- **用件 5 値**: 再診 / 予約変更 / 予約キャンセル / 初診 / 代表案内（初診/代表案内は END 直結）
- **再診ルート**: 用件 → 診療科聴取 → 予約候補日（フリー）→ 生年月日 → 氏名 → 連絡先 → END_受付完了_再診（smsFlag=1）
- **変更ルート**: 用件 → 診療科数確認 → 診療科 or 複数診療科（フリー）→ 予約日（現）→ 予約候補日（新）→ 病院への追加連絡事項 → 生年月日 → 氏名 → 連絡先 → END_受付完了_変更（smsFlag=2）
- **キャンセルルート**: 用件 → 診療科数確認 → 診療科 or 複数診療科（フリー）→ 予約日 → キャンセル理由 → 病院への追加連絡事項 → 生年月日 → 氏名 → 連絡先 → END_受付完了_キャンセル（smsFlag=3）
- **診療科 17 科 + 消化器分岐**: 「消化器」発話のみ消化器内科 / 消化器外科の二段聴取。他科は単発確定
- **個人情報サブフロー 3 本**: 生年月日聴取 / 氏名聴取 / 電話番号聴取（診察券番号は Gen1 にも無い）
- **代表電話 0972-22-0180**: 初診/代表案内/聴取失敗の 3 終話で 3 回繰り返し読み上げ
- **予約日聴取の DTMF 4 桁対応**: 「シャープまたはアスタリスク + MM(2)+DD(2)」を受付。比較日より前なら +1 年
- **echo_back 対象**: 用件 / 診療科 / 診療科数確認 / 予約日（現）の 4 ヶ所。消化器分岐や複数診療科聴取・自由テキスト聴取（理由・連絡事項・候補日）は echo_back なし
