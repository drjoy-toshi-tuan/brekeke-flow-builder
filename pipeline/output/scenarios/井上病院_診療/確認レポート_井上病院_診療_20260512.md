- **2 系統分岐型フロー**: 経路判断 hearing で WEB 受付希望（Yes）/ 電話受付希望（No）を判定 → 2 箇所（生年月日後・終話前）で context 値を再利用して CMR で分岐

- **電話ルートのみ問い合わせ内容聴取**: 自由テキスト hearing（OpenAI は summarize / 用件分類しない）
- **3 回失敗 → 代表案内 終話に統一**: 「予約制ではないので、基本的に当日受診が可能です。診療時間や休診情報等に関しては代表番号にてご確認ください。」
- **時間外 TTS**: 「申し訳ございません。ただいまの時間は受付時間外となっております。受付時間は、祝日を除く、月曜日から金曜日、朝8時30分から16時までです。恐れ入りますが、受付時間内におかけなおしください。」
- **DTMF 併用**: 診察券番号（発話 / DTMF 8 桁）、生年月日（発話 / DTMF 8 桁）、電話番号（発話 / DTMF）
