### 1.5 シナリオフロー（PDF記載）
```
非通知判定 → 冒頭アナウンス → 個人/企業
  個人 → 氏名 → 生年月日 → 電話番号 → 用件
  企業 → 企業名/団体名 → 担当者名 → 電話番号 → 用件

用件分岐:
  予約（個人） → 健保名 → 代行機関名 → 希望コース → オプション → 希望月・曜日 → 最終確認 → (FAQ) → 終話①/③
  予約（企業） → 健保名 → 代行機関名 → 受診者数 → 希望コース → 希望月・曜日 → 最終確認 → (FAQ) → 終話①/③
  変更 → 変更内容（日にち/それ以外）
    日にち → 現在の予約日 → 希望月・曜日 → 最終確認
    それ以外 → 詳細確認① → 現在の予約日 → 最終確認
  キャンセル → キャンセル内容（はい/いいえ）
    はい → 現在の予約日 → 次回予約希望有無
      あり → 希望月・曜日 → 最終確認
      なし → 最終確認
    いいえ → 詳細確認② → 現在の予約日 → 次回予約希望有無
  その他問い合わせ → 確認内容① → (複数あり → 確認内容② → 折り返し電話)
    折り返し はい → 終話③
    折り返し いいえ → 終話②
```
