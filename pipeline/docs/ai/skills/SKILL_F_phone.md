# SKILL_F: 電話番号型（参考保管）

## 用途

電話番号聴取サブフロー内のOpenAIモジュールで使用するプロンプト。電話番号の正規化・判定を行う。

## 使用条件

- 電話番号聴取サブフロー（`個人情報サブフロー.bivr` に含まれる）
- **重要**: このSkillは参考保管用。実際のプロンプトは `docs/reference/bivr/samples/個人情報サブフロー.bivr` からそのままコピーして使用する

## ベースプロンプト

ソース: `docs/reference/bivr/samples/個人情報サブフロー.bivr` > 電話番号聴取フロー内のOpenAIモジュール

**注意**: 個人情報サブフロー（氏名/生年月日/電話番号/診察券番号）のプロンプトはリファレンスbivrから完全コピーする運用ルールのため、このファイルにベースプロンプトは記載しない。

参照手順:
1. `docs/reference/bivr/samples/個人情報サブフロー.bivr` を展開（`python3 scripts/extract_bivr.py`）
2. 電話番号聴取フローのJSONから該当OpenAIモジュールの `params.prompt` を取得
3. フロー名プレフィックスのみ対象施設に置換し、プロンプト本文は一切変更しない

## カスタマイズ箇所

| 箇所 | 変更可否 | 説明 |
|---|---|---|
| フロー名プレフィックス | 変更可 | 対象施設名に置換 |
| プロンプト本文 | **変更不可** | リファレンスからそのままコピー |
| モジュール構造 | **変更不可** | next/subs/params構造はリファレンスに従う |

## 注意事項

1. **prompterはこのSkillを直接使用しない**: 個人情報サブフローのプロンプトはgeneratorがリファレンスからコピーする
2. **prompterがスキップする対象**: フロー名に「氏名聴取」「生年月日聴取」「電話番号聴取」「診察券番号聴取」を含むフローのOpenAIプロンプトはprompterの担当外
3. **このファイルの存在意義**: 電話番号型プロンプトのパターンを理解するための参考資料。新規パターン作成時のリファレンスとして使用可能
