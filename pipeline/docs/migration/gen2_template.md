# 第2世代プロンプト メタ情報テンプレート

第2世代プロンプトを `docs/migration/gen2_{施設名}_{シナリオ名}.txt` に配置する際、
ファイル先頭に以下のメタ情報をコメントとして記載することを推奨。

---

## テンプレート

```
# ============================================================
# 施設名: 〇〇病院
# シナリオ名: 診療予約
# 環境: デモ / 本番
# OpenAI Assistant ID: asst_XXXXXXXXXXXXXXXXXXXX
# 元ファイル場所: Google Drive > 案件フォルダ > プロンプト
# 取得日: YYYY-MM-DD
# 取得者: 担当者名
# 備考: 特記事項があれば記載
# ============================================================

{function定義JSONここから}
...
{function定義JSONここまで}

{プロンプト本文ここから}
...
{プロンプト本文ここまで}
```

---

## 記入例

```
# ============================================================
# 施設名: 羽生総合病院
# シナリオ名: 診療
# 環境: 本番
# OpenAI Assistant ID: asst_abc123xyz
# 元ファイル場所: Google Drive > 羽生総合病院 > AI電話 > プロンプト_v3.txt
# 取得日: 2026-03-20
# 取得者: 浜口
# 備考: 2026年3月更新版。診療科リストに甲状腺を追加。
# ============================================================

{
  "name": "dialogue_completed",
  ...
}

羽生総合病院 代表電話AI応答用プロンプト
...
```

---

## フィールド説明

| フィールド | 説明 | 必須 |
|---|---|---|
| 施設名 | 病院・クリニック名 | 必須 |
| シナリオ名 | 診療/受付/健診等 | 必須 |
| 環境 | デモ または 本番 | 必須 |
| OpenAI Assistant ID | 元の Assistant の ID | 推奨 |
| 元ファイル場所 | Google Drive のパス | 推奨 |
| 取得日 | プロンプト取得日 | 推奨 |
| 取得者 | 担当者名 | 推奨 |
| 備考 | バージョン履歴・特記事項 | 任意 |
