### 抽出した要件サマリー

- **入電者は他医療機関の連携担当者**（患者本人ではない）。入電者情報（医療機関名 / 担当者氏名 / 担当者連絡先）と患者情報（患者氏名 / 患者生年月日）を別フィールドで聴取
- **冒頭ゲート**: opening 直後に「緊急対応の有無」yes/no 判定。yes → 代表電話案内で即終話（status=2）
- **共通聴取の順序**: 医療機関名 → 入電者氏名 → 入電者電話番号 → 用件
- **用件 4 分岐**（classification）:
  - 外来入院申込み → 患者氏名 → 患者生年月日 → FAX送信時期 → END_終話1
  - 情報提供依頼 → 診療科（整形外科除外） → 患者氏名 → 患者生年月日 → 返信期限 → END_終話2
  - 入退院支援室 → 担当者名 → 患者氏名 → 患者生年月日 → END_終話2
  - その他お問い合わせ → 問い合わせ内容（「以上です。」待ち） → END_終話2
- **整形外科除外**: 情報提供依頼ルートで診療科が「整形外科」「整形」「せけげか」「せけいげか」等と判定された場合は AI 対応不可として代表電話案内で終話（status=2）
- **終話パターン 2 種**:
  - END_終話1: 外来入院申込み専用「確認事項がある場合のみ折り返し」
  - END_終話2: 情報提供依頼 / 入退院支援室 / その他共通「5 診療日以内にスタッフから連絡」
- **復唱**: 本番版プロンプト準拠で **全項目復唱オフ**（echo_back=false）
- **再質問**: 本番版プロンプト準拠で **最大 2 回**、2 回失敗で END_聴取失敗（代表電話案内）
- **営業時間**: 月〜土 8:30〜12:00（日曜祝日・年末年始除く）
