  - 冒頭: 非通知即時終話（incoming-classifier）

  - 申込ルート内: 胃カメラ有無で質問順切替 Pattern A 分岐（有→胃カメラ方法→予約希望日→コース→会社名 / 無→コース→予約希望日→会社名）
  - 終話: classification × 電話番号種別（携帯/固定）の2段階 Pattern B 分岐
- **終話パターン**（正常系3本＋異常系3本）:

- **復唱ルール**: 生年月日のみ「○○ですね」1回許可。氏名・電話番号・コース・予約日・変更内容は全て禁止
- **連絡先電話番号**: 聴取しない。`incoming_phone_number`（着信番号）のみを連絡先として使用
- **RAG/FAQ**: 未使用。プロンプト外の質問は再質問 1回→代表電話案内で終話
