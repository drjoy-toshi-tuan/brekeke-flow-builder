### 抽出した要件サマリー

- **冒頭挨拶**: 「こちらは多摩総合医療センターの、予約専用、AI電話です。」+「紹介状や、診察券をお持ちの方はお手元にご用意ください。」
- **受付時間**: 日曜祝日除く月〜土 8:30〜17:00。年末年始（12/28〜1/4）は時間外。時間外文言には WEB フォーム誘導を含む
- **用件分類**: 6 値（診察 / 再診 / 変更 / キャンセル / 確認 / その他）。「診察」は新規予約（初診含む）の意味、「再診」は再診予約
- **初診ルート**: 診察 → 受診歴確認 → 受診歴なし: 紹介状聴取（紹介状なし→END）→ 紹介元医療機関聴取 → 紹介元電話番号聴取 → 初診_診療科（特殊診療科分岐多数）→ 個人情報聴取
- **再診ルート**: 再診 OR 診察+受診歴あり → 前回症状確認 → 再診_診療科（特殊診療科分岐）→ 個人情報聴取（診察券あり）
- **変更/キャンセルルート**: 診療科（精神神経科/産婦人科特殊）→ 理由 → 予約日 → 個人情報聴取（診察券あり）
- **確認ルート**: 診療科 → 個人情報聴取（診察券あり）
- **その他ルート**: 「予約センター 042-323-9200 / 代表電話 042-323-5111」案内 → END
- **診療科特殊分岐マトリクス**:
  - 初診のみ受付不可（医療連携室誘導）: ゲノム診療科 / 腫瘍内科 / 血液内科 / 緩和ケア内科
  - AI電話受付不可（予約センター誘導）: リハビリテーション科 / 看護外来
  - 受付停止中: 女性専用外来（初診・再診とも不可）
  - 別機関誘導: 小児科（小児総合医療センター・地域医療機関へ）
  - 代表電話誘導（初診）: 救急診療科 / 救命・集中治療科
  - 代表電話誘導（再診）: 救急 / 救命 / 腫瘍内科 / 緩和ケア内科
  - 紹介元医療機関聴取に直接遷移: リウマチ外科 / リウマチ膠原病内科（初診ルートのみ）
  - 変更/キャンセル特殊（後ほど折返し）: 精神神経科 / 産婦人科（妊婦検診）
- **電話番号サブフロー**: phonetype を 1=携帯/ip050、2=その他（landline/withheld）として保存
- **終話分岐**: classification × phonetype の 2 段 ContextMatchRouter
- **終話TTS**: 用件ごとに「{用件}の申し込みを受付いたしました。」+「本日を含めて3営業日以内に担当者から折り返し電話、もしくはショートメッセージにてご連絡いたします。」（携帯/固定で SMS 言及を出し分け）
