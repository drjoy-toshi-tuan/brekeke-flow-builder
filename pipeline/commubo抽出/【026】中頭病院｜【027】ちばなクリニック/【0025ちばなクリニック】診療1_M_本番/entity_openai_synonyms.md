# OpenAI Synonym Examples — 中頭病院 / 診療

## わからない

- **わからない**: 覚えて, 分からない, 分らない, 覚えてない, 知らない, 知らね, 知らん, 分からん

## 別の

- **別の**: 違う, 携帯, 他の, 変え

## 否定

- **いいえ**: そうじゃないねん, そうちゃうねん, まちがった

## 否定単語

- **違います**: 違い, 違う, 違って, 駄目

## 否定（エンティティ）

- **否定**: 持ってない, 持ってません。

## 希望しない

- **希望しない**: 結構, 駄目, 無理

## 希望する

- **希望する**: 可能, 見たい, 見る

## 持ってない

- **持ってない**: 以てない, 持ってませーん, 持ってません, 盛ってない

## 用件

- **キャンセル**: 2, 2番, ニ番, 取り消す, 二、
- **確認**: 3, 3番, その他確認, 角煮, 教えて, 三, 三番, 時間を忘れた, 日時確認, 聞きたい, 約確認, 予約の核, 予約の確認, 予約確認, 予約日時を確認する
- **変更**: 1, 1番, 一, 一番, 延期, 取り直したい, 取り直す, 判子, 平行, 並行, 変えて, 変える

## 肯定

## 肯定単語
