# OpenAI Synonym Examples — 医誠会国際総合病院 / 施設間搬送

prompter は generate_by_OpenAI モジュールの prompt の `## 類義語例` 等のセクションに反映する。

## もう一回、間違えた

- **間違え**: もう１回, もう一回, もう一度, 間違い
- **間違えた**: もう１回, もう一回, もう一度

## もう一度

- **もう一度**: もう一回

## 別の

- **別の**: 違う, 携帯, 他の, 変え

## 否定

- **否定**: 違う, 間違い

## 否定単語

- **違います**: 違い, 違う, 違って, 駄目

## 持ってない

- **持ってない**: 以てない, 持ってませーん, 持ってません, 盛ってない

## 救急搬送希望

- **1**: 1番, 一番, 救急, 搬送
- **搬送希望**: 1, 1番, 一番, 救急, 緊急, 搬送

## 直接来院

- **2**: 行きます, 行く, 自分で, 直接, 二番
- **直接来院**: 2, 行きます, 行く, 自分で, 直接, 二番

## 肯定

- **肯定**: 1, 1番, お願いします, 一番, 必要です

## 肯定単語

- **はい**: 違いありません, 違いない, 大丈夫, 問題ありません, 問題ない

- **間違い**: 違い, 違う, 間違え
