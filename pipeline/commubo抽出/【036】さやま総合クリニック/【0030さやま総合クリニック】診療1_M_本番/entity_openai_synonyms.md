# OpenAI Synonym Examples — さやま総合クリニック / 診療

Gen1 commubo エンティティ定義から抽出した、OpenAI 正規化層で吸収すべき類義語例。
prompter は generate_by_OpenAI モジュールの prompt の `## 類義語例` 等のセクションに反映する。

## もしもし

- **もしもし** (synonyms: なし)

## 今日

- **今日**: 今から, 当日, 当日予約

## 代表案内案件

- **代表案内** (synonyms: なし)

## 分からない

- **分からない**: 解らない, 持ってない, 動かない, 分かりません

## 別の

- **別の**: 違う, 携帯, 他の, 変え

## 否定

- **否定**: 違う, 間違い, 間違っている, 間違ってます, 間違って, 間違ってる
- **いいえ**: そうじゃないねん, そうちゃうねん, ちがう, まちがった

## 否定単語

- **違います**: 違い, 違う, 違って, 駄目

## 持ってない

- **持ってない**: 以てない, 持ってませーん, 持ってません, 盛ってない

## 用件

- **キャンセル**: 3, 3番, 三, 三番, 取り消す
- **定期予約**: 1, 1番, 一, 一番, 最初, 最新, 撮りたい, 三新薬, 始め, 取得, 受診, 診察, 診察の予約, 診療予約, 予約お願い, 予約する, 予約をしたい, 予約をする, 予約をとる, 予約を撮り, 予約を撮る, 予約を取り, 予約を取る, 録りたい, 録る
- **変更**: 2, 2番, 延期, 取り直したい, 取り直す, 二番, 判子, 平行, 並行, 変えて, 変える

## 肯定

- **肯定**: 大丈夫です, 正しい, 異常部, 丈夫, 大丈夫, 配送です

## 肯定単語

- **はい**: 違いありません, 違いない, 大丈夫, 問題ありません, 問題ない
