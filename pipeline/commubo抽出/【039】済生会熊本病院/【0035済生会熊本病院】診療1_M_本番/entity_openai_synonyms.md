# OpenAI Synonym Examples — 済生会熊本病院 / 診療

Gen1 commubo エンティティ定義から抽出した、OpenAI 正規化層で吸収すべき類義語例。
prompter は generate_by_OpenAI モジュールの prompt の `## 類義語例` 等のセクションに反映する。

## もしもし

- **もしもし** (synonyms: なし)

## わからない

- **わからない**: 覚えてない, 書いていない, 知らない, 分からない, 分からん, 忘れちゃった

## 分からない

- **分からない**: 解らない, 持ってない, 動かない, 分かりません

## 別の

- **別の**: 違う, 携帯, 他の, 変え

## 否定

- **いいえ**: そうじゃないねん, そうちゃうねん, ちがう, まちがった

## 否定単語

- **違います**: 違い, 違う, 違って, 駄目

## 持ってない

- **持ってない**: 以てない, 持ってませーん, 持ってません, 盛ってない

## 用件

- **キャンセル**: 3番, よく県する, 行けない, 三番, 取り消す, 二百株, 美白の検査, 予約権せる, 余白よく検査
- **その他**: 4番, お問合せ, その他おといあわせ, その他お問い合わせ, 教えて, 四番
- **確認**: 1番, の確認, を確認, 一番, 確認したい, 確認の希望, 確認を希望, 確認希望
- **変更**: 2番, いや抗原航空, の銀行, の変更, の辺, 延期, 取り直したい, 取り直す, 二番, 判子, 平行, 並行, 変えて, 変える

## 肯定

- **肯定**: 異常部, 丈夫, 大丈夫, 配送です

## 肯定単語

- **はい**: 違いありません, 違いない, 大丈夫, 問題ありません, 問題ない
