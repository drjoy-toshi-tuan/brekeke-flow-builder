# OpenAI Synonym Examples — 新宿健診プラザ / 健診

Gen1 commubo エンティティ定義から抽出した、OpenAI 正規化層で吸収すべき類義語例。
prompter は generate_by_OpenAI モジュールの prompt の `## 類義語例` 等のセクションに反映する。

## もしもし

- **もしもし** (synonyms: なし)

## エンティティ保存

- **問い合わせ**: お伺いしたい, お問合せ, その他を取り合わせ, 確認したい, 教えて, 金額, 検査結果, 質問, 質問したい, 調べて, 聞きたい, 問い合わす, 問い合わせる, 料金

## 分からない

- **分からない**: 解らない, 持ってない, 動かない, 分かりません

## 別の

- **別の**: 違う, 携帯, 他の, 変え

## 否定

- **否定**: 違う, 間違い, 間違って, 間違えた, 間違えました, 間違った

## 否定単語

- **違います**: 違い, 違う, 違って, 駄目

## 持っている

- **持っている**: 持っていまーす, 持ってまーす, 持ってます, 盛ってまーす, 盛ってます

## 持ってない

- **持ってない**: 以てない, 持ってませーん, 持ってません, 盛ってない

- **肯定**: 大丈夫です, 異常部, 丈夫, 大丈夫
