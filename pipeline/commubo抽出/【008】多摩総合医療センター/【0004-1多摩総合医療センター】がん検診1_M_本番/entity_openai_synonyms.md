# OpenAI Synonym Examples — 多摩総合医療センター / がん検診

Gen1 commubo エンティティ定義から抽出した、OpenAI 正規化層で吸収すべき類義語例。
prompter は generate_by_OpenAI モジュールの prompt の `## 類義語例` 等のセクションに反映する。

## もう一度

- **もう一度**: もう1度, もう一回, 再度, 1回, 1度, 一回, 一度

## わからない

- **わからない**: 知らない, 分からない, 分かりません

## 分からない

- **わからない**: 知らない, 分かりません
- **分からない**: 分かりません, 解らない, 持ってない, 動かない

## 別の

- **別の**: 違う, 携帯, 他の, 変え

## 否定

- **何度**: 何度かあります, 何度かある
- **否定** (synonyms: なし)

## 否定単語

- **違います**: 違い, 違う, 違って, 駄目

## 持ってない

- **持ってない**: 以てない, 持ってませーん, 持ってません, 盛ってない

## 肯定

- **はい**: その通り, 受けたことない, 受け取っている, 受け取ってます, 受け取ってる

## 肯定単語

- **はい**: 違いありません, 違いない, 大丈夫, 問題ありません, 問題ない
