# OpenAI Synonym Examples — 井上病院 / 発熱外来

Gen1 commubo エンティティ定義から抽出した、OpenAI 正規化層で吸収すべき類義語例。
prompter は generate_by_OpenAI モジュールの prompt の `## 類義語例` 等のセクションに反映する。

## もう一度

- **もう一度**: もう一_回

## 分からない

- **分からない**: 解らない, 持ってない, 動かない, 分かりません

## 別の

- **別の**: 違う, 携帯, 他の, 変え

## 否定

- **いいえ**: 違う, 間違えた, 間違った

## 否定単語

- **違います**: 違い, 違う, 違って, 駄目

## 性別

- **女性**: 助成, 女
- **男性**: 弾性, 男

## 持ってない

- **もってない**: 手元にありません
- **持ってない**: 以てない, 持ってませーん, 持ってません, 盛ってない

## 肯定

- **はい**: その通りです, 異常部, 丈夫, 大丈夫, 配送です

## 肯定単語

- **はい**: はい大丈夫です, 違いありません, 違いない, 大丈夫, 配送です, 問題ありません, 問題ない
