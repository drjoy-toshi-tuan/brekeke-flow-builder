# OpenAI Synonym Examples — 湘南記念病院 / 診療

Gen1 commubo エンティティ定義から抽出した、OpenAI 正規化層で吸収すべき類義語例。
prompter は generate_by_OpenAI モジュールの prompt の `## 類義語例` 等のセクションに反映する。

## もう一度

- **もう一度**: もう一_回

## もしもし

- **もしもし** (synonyms: なし)

## わからない

- **わからない**: 覚えてない, 書いていない, 知らない, 分からない, 分からん, 忘れちゃった

## 分からない

- **分からない**: 解らない, 持ってない, 動かない, 分かりません

## 別の

- **別の**: 違う, 携帯, 他の, 変え

## 否定

- **いいえ**: いーえ, まちがった, そうじゃないねん, そうちゃうねん
- **否定**: EHが今

## 否定エンティティ

- **否定**: EHが今

## 否定単語

- **違います**: 違い, 違う, 違って, 駄目

## 持ってない

- **持ってない**: 以てない, 持ってませーん, 持ってません, 盛ってない

## 用件

- **キャンセル**: 3, 3番, よく県する, 行けない, 三, 取り消す
- **確認**: 4, 4番, の確認, を確認, 確認したい, 確認の希望, 確認を希望, 確認希望, 四
- **診察**: 1, 1番, 一, 再診の予約, 四百とります, 信託万二百オフ後, 診察の予約, 薬をとる, 予約をお願い, 予約をポール, 予約を取りたい, 予約を取る, 余白を取る
- **変更**: 2, 2番, いや抗原航空, の銀行, の変更, の辺, 延期, 取り直したい, 取り直す, 二, 二番, 判子, 平行, 並行, 変えて, 変える

## 肯定

- **はい**: その通りです, 異常部, 丈夫, 大丈夫, 配送です
- **肯定**: 異常部, 丈夫, 大丈夫, 配送です, 合って

## 肯定エンティティ

- **肯定**: 異常部, 合って, 丈夫, 大丈夫, 配送です

## 肯定単語

- **はい**: 違いありません, 違いない, 大丈夫, 問題ありません, 問題ない
